# App para controle de despesas e dicas sobre investimentos 

## Get started

1. Instalar dependencias

   ```bash
   npm install
   ```

2. Iniciar o APP local

   ```bash
    npx expo start
    ou
    npm start
   ```

3. Buildar o APP

   ```bash
   eas build --profile production --platform android
   ```
